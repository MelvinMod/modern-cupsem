# современный-cupsem

Консорциум программного обеспечения для высшего уровня физики создал программный продукт «Симуляции электричества и магнетизма» (CUPSEM), который был выпущен в 1995 году и использует интерактивные симуляции, чтобы помочь пользователям понять концепции электричества и магнетизма.  Он содержит ряд ресурсов и инструментов для обучения физике.  Это приложение было декодировано и вручную переписано на Ruby для современных Linux и Windows 10 мной, MelvinSGjr (известным как MelvinMod (мой второй аккаунт, так как предыдущий не имеет доступа к постам)).  **Всё было переписано вручную на Ruby, без использования искусственного интеллекта, но искусственный интеллект использовался только для понимания исходного кода декомпилированного приложения!!!**

### Используемые основные гемы
- **Numo::Narray**  - Числовые массивы и операции с матрицами
- **Numo::Gnuplot**  - 2D/3D построение графиков (через gnuplot)
- **RubyInline** или **FFI**  - Для производительного кода
- **Racc** - Генерация парсера выражений
- **Ruby/GTK3** или **Shoes**  - GUI-фреймворк

## Статус
Это работа в процессе.  Исходный код на Pascal был проанализирован, и разрабатывается порт на Ruby.

## Как установить
Установите рабочий язык программирования Ruby (и Chocolatey для Windows).

**Текущее состояние**: Код работает в терминальном/текстовом режиме (только ввод текста). 
**Чтобы получить настоящую графику (как в оригинальном CUPS)**: Установите Gnuplot:
```bash
# На EndeavourOS (Arch Linux)
sudo pacman -S gnuplot
```
```powershell
# На Windows 10/11 (используя Chocolatey)
choco install gnuplot
```
# На Windows 10/11 (используя Chocolatey)
choco install gnuplot
```

После установки gnuplot вы можете:

- Построить 2D функции с кривыми
- Построить 3D поверхности
- Построить контурные карты
- Построить векторные поля
- Визуально отображать распределения зарядов

**Как бегать:**

```bash
# Сначала выполните настройку
ruby bin/setup.rb

# Запустить главное меню
ruby -Ilib lib/cupsem/main.rb

# Запустить конкретную симуляцию
ruby -Ilib lib/cupsem/main.rb --gauss

# Показать справку
ruby -Ilib lib/cupsem/main.rb --help
```

## Что работает:
-  Математические функции (pwr, sin, cos, tan, sinh и т.д.) 
-  Операции с матрицами

- Парсер выражений

- Численное интегрирование (Симпсона, трапеций, Гаусса)
- Поиск корней
- Интерполяция 
- Базовые симуляции (Поля, Гаусс)
