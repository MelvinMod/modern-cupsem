# современный-cupsem
Консорциум программного обеспечения для высшего уровня физики создал программный продукт «Моделирование электричества и магнетизма» (CUPSEM), который был выпущен в 1995 году и использует интерактивные симуляции, чтобы помочь пользователям понять концепции электричества и магнетизма. Он содержит ряд ресурсов и инструментов для обучения физике. Это приложение было декодировано и вручную переписано на Ruby для современных Linux и Windows 10 мной, MelvinSGjr (известным как MelvinMod (мой второй аккаунт, так как предыдущий не имеет доступа к постам)). **Всё было переписано вручную на Ruby, без использования искусственного интеллекта, но искусственный интеллект использовался только для понимания исходного кода декомпилированного приложения!!!**

### Используемые основные гемы
- **Numo::Narray** - Числовые массивы и операции с матрицами
- **Numo::Gnuplot** - 2D/3D построение графиков (через gnuplot)
- **RubyInline** или **FFI** - Для кода, критичного к производительности
- **Racc** - Генерация парсера выражений
- **Ruby/GTK3** или **Shoes** - GUI-фреймворк

## Статус
Это работа в процессе. Исходный код на Pascal был проанализирован, и разрабатывается порт на Ruby.
